Gestione Ri.Ba. / Creazione di file CBI per flussi RiBa

Create CBI file for RiBa.

For the module to work the SIA code of the company must be set in "Settings" -> "General Settings" -> "Accounting" under the "RiBa CBI" section.

Creazione di file CBI per flussi RiBa

Il modulo richiede che il codice SIA dell'azienda sia impostato in "Impostazioni" -> "Impostazioni Generali" -> Contabilità" sezione "Riba CBI".
