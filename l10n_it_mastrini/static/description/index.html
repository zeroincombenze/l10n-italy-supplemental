Mastrino contabile

:alt: l10n_it_mastrini :width: 98%

This software show dynamically the account statement with initial, progressive e final balance.

Questo moduli permette di visualizzare dinamicamente i mastrini contabili. Visualizza il saldo iniziale, progressivo e finale del conto.
