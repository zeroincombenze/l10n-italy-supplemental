Italian Localisation - Natura delle aliquote IVA / N/D

This module extends the functionality of account tax.

With this module is possibile to set a kind of exemption for every account tax you need.

N/D

Usage / Utilizzo

To use this module, you need to:

#. Go to Accounting -> Configuration -> Accounting -> Taxes #. Open the tax to modify, go to "Advanced Options" tab and set an Exemption Kind value
